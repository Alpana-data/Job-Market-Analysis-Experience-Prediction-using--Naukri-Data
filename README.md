# Overview:
To Analyze job market trends to identify in-demand skills and predict experience levels required for the roles.
# Objective:
To understand hiring patterns, skill demand, and build a predictive model for experience level classification.
# Tools Used:
MS Excel | Power BI | Python | Machine Learning
# Dataset:
Source: Naukri.com
Tools used for Web Scraping: Selenium, BeautifulSoup4
Number of pages scraped: 100 pages
The dataset contains job-related information such as job title, location, company, reviews, ratings, required skills, and experience levels
# Tasks Performed: 
Webscraping, Data preprocessing, Exploratory Data Analysis, Visualizations, Prediction model create/
# Key Insights:
Skill Demand:  Data Science, Python, and Machine Learning are the most in-demand skills.
Experience Distribution: Most job opportunities are concentrated in mid and senior-level roles, with fewer entry-level openings.
Location Trend: Job demand is highly concentrated in cities like Bengaluru, Hyderabad, and Gurugram.
Company Ratings: Most companies fall within medium to high rating segments, indicating generally positive employer feedback.
Model Performance: The classification model achieved around 65% accuracy, with better performance for senior-level roles.
